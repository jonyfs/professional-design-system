Chart

Line, Bar, Area, Pie/Donut, Radar, and Radial charts, plus a shared tooltip and legend — all six re-color automatically on every theme preset and ship with a non-visual data-table equivalent for screen reader users.

This is the one feature in this catalog that ships React-only (specs/020-recharts-chart-primitives/spec.md, Assumptions): Recharts has no framework-independent rendering path, so there is no zero-JavaScript static version of this component to demo here.
