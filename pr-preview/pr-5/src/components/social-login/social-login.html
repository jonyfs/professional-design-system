Social Login Buttons

A configurable group of brand-compliant sign-in buttons. The 5 built-in presets (Google, Apple, Facebook, Microsoft, GitHub) match each provider's own published button guideline by default; the same shape accepts a custom entry for any other provider. Presentation only — no OAuth flow of its own (each activation is the host application's own responsibility).

Built-in presets (US1)

The GitHub button above illustrates the independent per-provider disabled/loading state (US2, FR-005) — every other button remains fully interactive.

Custom entries — no mandated spec (US3)

Every custom entry shares the presets' exact height, padding, and radius (US3, FR-009) — the supplied brand color styles only the icon's accent backing, never the button surface or text (research.md R1/R5).
