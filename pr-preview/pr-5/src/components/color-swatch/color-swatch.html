Color Swatch

Color chip whose background is the caller-supplied value. Because the value is unbounded, it is applied via data-swatch-color through the CSSOM (keeping the strict style-src 'self' CSP), and an sr-only label carries the color name/value so it is never conveyed by hue alone.

  • Success green, #16a34a Success green
