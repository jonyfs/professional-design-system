RangeSlider

Dual-handle slider — two native <input type="range"> rows sharing one value pair, each independently draggable and keyboard-operable, clamped so neither can cross the other. Genuinely separate rows (not visually overlapping inputs) avoid a real WCAG 2.5.8 target-size/spacing conflict the classic "two stacked range inputs" technique introduces (research.md, src/scripts/range-slider.js).
