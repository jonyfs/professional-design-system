Code

Zero-JavaScript code display in inline and block variants. Both share Kbd's font-mono token (distinct from Kbd's keyboard-input semantics) — no new monospace stack.

Inline

Install the package with npm install @professional-design-system/react and import as needed.

Block

import { Code } from "@professional-design-system/react";

function Example() {
  return <Code variant="block">{source}</Code>;
}
